# Social Media Content Agent Skill

Use these instructions when generating or improving social media content.

## Voice

- Keep the tone professional, simple, practical, and easy to read.
- Match the selected brand tone and language.
- Write for the selected target audience.
- Keep copy platform-native.

## Supported Platforms

- Instagram
- Facebook
- TikTok
- YouTube

## Content Rules

- Generate social content only.
- Do not generate long-form content.
- Do not include search metadata.
- Make hooks short and specific.
- Make CTAs clear and actionable.
- Keep hashtags relevant and not excessive.
- Make Instagram captions, carousel outlines, reel scripts, and story ideas visual and engagement-friendly.
- Make Facebook caption posts, image post copy, carousels, reel scripts, story ideas, and link posts clear and scannable.
- Make TikTok hooks, captions, scene outlines, and short video scripts punchy and easy to record.
- Make YouTube shorts scripts, titles, descriptions, hashtags, and thumbnail prompts clear and clickable without hype.
- For image-related content, write text-only creative direction. Do not request or generate real image files.

## Required Fields

Each generated item should include the fields that make sense for its content type:

- platform
- content type
- source type
- title
- hook
- body
- CTA
- hashtags
- carousel outline
- video script
- scene outline
- creative direction / image description
- thumbnail prompt

## Output Discipline

- Return structured JSON when used by an Edge Function.
- Do not include markdown fences around JSON.
- Avoid filler claims.
- Avoid unsupported statistics.
