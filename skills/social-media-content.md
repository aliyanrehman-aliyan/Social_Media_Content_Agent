# Social Media Content Agent Skill

Use these instructions when generating or improving social media content.

## Voice

- Keep the tone professional, simple, practical, and easy to read.
- Match the selected brand tone and language.
- Write for the selected target audience.
- Keep copy platform-native.

## Supported Platforms

- Instagram
- Facebook
- TikTok
- YouTube

## Content Rules

- Generate social content only.
- Do not generate long-form content.
- Do not include search metadata.
- Before generating, use available live SERP context to understand current trends, competitor angles, common hooks, and market gaps.
- Use competitor context as inspiration only. Never copy competitor wording, names, captions, or claims.
- Make every output fit the selected platform and the selected content type exactly.
- Make hooks short and specific.
- Make CTAs clear and actionable.
- Keep hashtags relevant and not excessive.
- Make Instagram captions, carousel outlines, reel scripts, and story ideas visual and engagement-friendly.
- Make Facebook caption posts, image post copy, carousels, reel scripts, story ideas, and link posts clear and scannable.
- Make TikTok hooks, captions, scene outlines, and short video scripts punchy and easy to record.
- Make YouTube shorts scripts, titles, descriptions, hashtags, and thumbnail prompts clear and clickable without hype.
- For asset/image-related content, write text-only Asset Ideas. Do not request or generate real image files.

## Platform Asset Rules

- Instagram assets should feel native to feed posts, carousel slides, Stories, Reel covers, quote cards, and product or lifestyle visuals.
- Facebook assets should feel native to feed posts, carousels, banners, link previews, community posts, and event-style visuals.
- TikTok assets should feel native to vertical short video, creator-style opening frames, hook scenes, text overlays, scene outlines, and fast thumb-stop visuals.
- YouTube Shorts assets should feel native to vertical Shorts thumbnails, opening scenes, hook visuals, end screens, titles, and descriptions.
- Each Asset Idea must include visual concept, main subject or object, background style, text overlay idea, mood/style, composition, and platform framing.

## Required Fields

Each generated item should include the fields that make sense for its content type:

- platform
- content type
- source type
- title
- hook
- body
- CTA
- hashtags
- carousel outline
- video script
- scene outline
- asset idea / image description
- thumbnail prompt

## Output Discipline

- Return structured JSON when used by an Edge Function.
- Do not include markdown fences around JSON.
- Avoid filler claims.
- Avoid unsupported statistics.
